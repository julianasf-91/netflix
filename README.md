# 🎬 Clone da Página de Login da Netflix

Este projeto é uma **réplica da página de login da Netflix**, desenvolvida com o objetivo de **praticar conhecimentos de Front-End**, especialmente estruturação com **HTML5** e estilização com **CSS3**.

A proposta foi reproduzir visualmente a interface da plataforma, exercitando conceitos importantes de desenvolvimento web, como organização semântica, posicionamento de elementos, uso de imagens, formulários e responsividade.

---

## 🚀 Objetivo do Projeto

O principal objetivo deste projeto foi colocar em prática fundamentos de **desenvolvimento Front-End**, recriando uma interface conhecida do mercado para treinar:

- Estruturação de páginas com HTML
- Estilização com CSS
- Organização visual de layout
- Construção de formulários
- Uso de imagens e ícones
- Boas práticas de marcação semântica

---

## 🖥️ O que foi desenvolvido

A aplicação reproduz a tela de login da Netflix, contendo:

- Logo da plataforma
- Imagem de fundo
- Formulário de login com:
  - campo de email ou telefone
  - campo de senha
  - botão de entrar
- Checkbox “Lembre-se de mim”
- Link de ajuda
- Opção de conexão com Facebook
- Texto para cadastro de novos usuários
- Rodapé com links institucionais
- Seletor de idioma

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
